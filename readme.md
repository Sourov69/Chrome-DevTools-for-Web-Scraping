# 🔍 Chrome DevTools for Web Scraping

This repository documents my learning journey of using Chrome DevTools for web scraping.

I completed a crash course from FreeCodeCamp and practiced key concepts such as DOM inspection, network analysis, and API discovery — essential skills for extracting data from modern websites.

---

## 📌 Learning Source

- FreeCodeCamp Crash Course  
  https://youtu.be/gTVpBbFWry8

---

## 📌 Project Objective

- Understand how websites load and display data
- Learn how to inspect HTML and DOM structure
- Identify hidden APIs using the Network tab
- Analyze request and response payloads
- Build a strong foundation for advanced web scraping

---

## 🛠 Tools Used

- Google Chrome DevTools
- Browser Network Tab
- Elements (DOM Inspector)
- Fetch/XHR Analysis

---

## 🔬 Key Concepts Learned

### 🔎 DOM Inspection
- Difference between **View Source vs Inspect**
- Understanding rendered HTML (JavaScript content)
- Identifying elements for scraping

### 🌐 Network Analysis
- Using Network tab to track requests
- Filtering Fetch/XHR requests
- Understanding headers, payload, and responses

### 🔑 API Discovery
- Finding hidden APIs behind websites
- Analyzing JSON responses
- Understanding tokens and request parameters

### ⚙️ Request Understanding
- GET vs POST requests
- Headers and authentication basics
- Query parameters and payload structure
